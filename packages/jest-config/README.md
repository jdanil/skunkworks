# `jest-config`

Sharable [jest config](https://jestjs.io/docs/en/configuration#preset-string).

## Usage

`.jestrc.json`

```json
{
  "preset": "@package/jest-config"
}
```

## Configs

### Default

`@package/jest-config`

Use with any project.

### Development

`@packages/jest-config/presets/development`

Use during development.
